## [v0.5.0] - 2026-04-05

### Highlights

- Audience-aware JWT tokens for BFF vs API client separation
- Ent-backed stores for API keys and BFF sessions with AES-256-GCM encryption
- Reusable Ent schema mixins for API keys and BFF sessions
